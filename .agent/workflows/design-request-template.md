# 🎨 デザインリクエストテンプレート

## 基本方針（常に適用）

以下は全てのデザインリクエストに適用される基本方針です：

```
## 🔒 必須要件

### デザインスタイル
- **Notionライクなシンプル・クリーンなデザイン**を採用
- 余白を十分に取り、情報を詰め込みすぎない
- 視覚的なノイズを最小限に抑える
- 機能性とミニマリズムを両立

### テーマ切替
- **ダークモード / ライトモード切替機能は必須**
- トグルボタンまたはシステム設定に追従
- 両モードで美しく見えるカラー設計

### タイポグラフィ
- システムフォント or モダンなサンセリフ（Inter, Noto Sans JP）
- 適切な行間・文字間
- 見出しと本文の明確な階層

### カラー
- 控えめで洗練されたカラーパレット
- アクセントカラーは最小限に効果的に使用
- ダークモードでは目に優しい色調
```

# 📝 AIへの指示文（コピペ用）

以下をそのまま使用し、`【】`内を埋めてください：

```markdown
## デザイン指示

### 基本スタイル
- Notionのようなシンプル・クリーンなデザイン
- ミニマルで余白を活かしたレイアウト
- 視覚的なノイズを排除

### 必須機能
- ダークモード / ライトモード切替（トグル必須）
- 両モードで美しく動作すること

### カラー
- 控えめで洗練されたパレット
- ダーク: 背景 #1e1e1e〜#2d2d2d、テキスト #e0e0e0
- ライト: 背景 #ffffff〜#f7f7f7、テキスト #37352f
- アクセント: 青系（#2383e2）or 好みで調整

### タイポグラフィ
- フォント: Inter + Noto Sans JP
- 明確な見出し階層
- 適切な行間・余白

### アプリ情報
- 名前: 【アプリ名】
- 目的: 【何をするアプリか】
- 画面構成: 【必要な画面をリスト】

### NGポイント
- 派手・ケバいデザイン
- 原色ベタ塗り
- ごちゃごちゃした情報過多
- 古臭い・バタ臭いUI
```

# 💡 実際の使用例

```markdown
Notionライクなシンプルデザインで、ダークモード切替付きの「タグピッカー」を作成してください。

DanbooruのURLから画像とタグを取得し、選択したタグをクリップボードにコピーできるツールです。

必要な画面:
- URL入力エリア
- 画像プレビュー
- カテゴリ別タグ一覧（チェックボックス付き）
- 選択タグ表示 & コピーボタン

追加要件:
- タグにホバーで日本語訳を表示
- カテゴリごとの一括選択ボタン
```
